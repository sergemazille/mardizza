#Mardizza

Application de gestion du choix de la pizza du mardi

## Dépendances
- Composer
- npm
- Gulp (global)

## Installation
- git clone https://github.com/sergemazille/mardizza.git
- cd mardizza
- composer install
- npm install
- gulp deploy

N'oubliez pas d'autoriser le répertoire `var/` en écriture.
